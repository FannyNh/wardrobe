# MVP Feature Set

## Frontend MVP Features:

User Authentication (optional for MVP, but useful for user-specific data):

- Simple login page (email/password).
- Basic user profile.

Basic Wardrobe Management:

- Upload clothes manually by entering their name/type.
- View a list of clothes in a basic table format.

Basic Outfit Generation:

- Hardcoded outfit suggestions based on wardrobe items (for now, just some static recommendations).

Basic UI Layout:

- Simple and clean UI using Vue.js and TailwindCSS to handle styling.

---

## Backend MVP Features:

User API:

- Simple REST API to create and authenticate users.

Wardrobe API:

- CRUD operations for managing wardrobe items (create, read, update, delete clothes).

Outfit API:

- Basic endpoint for suggesting outfits (even if static or rule-based for the MVP).
