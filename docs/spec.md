# Spec

`Hi ! i'm a developper and i want to create a new webapp using AI with vide coding with codex. BUT before that i need to prepare all the spec. please give me the list of thing's that need to be done.`

---

## Purpose and Use Cases

### Core Objective
 - inventory of the wardrobe
 - create outfit base on the wardrobe
 - propose clothe to buy 

### Target Users

- people who love fashion
- people working in the textile industry

### Key Features
- add clothe in e-wardrobe
- can select the type on clothe
- AI-driven proposition of outfit 
- AI generated avatar with the outfit

---

## Technical Requirements

### Tech Stack

- Frontend : vue.js (vue 3), pinia (state management),vite(builder), vitest(testing), Mock Service Worker (mock data), Tailwind(style), eslint/prettier(formating), typescript(Oriented object), sonar
- Backend : java 21, spring boot,mapstruct, lombok, postgresql, maven, mockito
- AI : codex

### Dev rules 

`Follow the principles of clean code and maintain a high level of readability.`

- Respect DDD method for backend and if possible fontend
- Meaningful Names
- Functions Should Do One Thing
- Keep Functions Small
- Avoid Side Effects
- Avoid Magic Numbers
- Single Responsibility Principle
- Open/Closed Principle
- Avoid Duplication
- Error Handling
- Avoid Long Method Chains
- Use Design Patterns
- Encapsulate What Varies
